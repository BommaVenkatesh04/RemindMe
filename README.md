# ⏰ RemindMe - Task Manager App

RemindMe is a modern React-based task management application designed to help users efficiently organize daily tasks with smart reminders, alarm notifications, snooze functionality, and productivity tracking.

## ✨ Features

- 📝 Create and manage daily tasks
- ⏰ Schedule tasks with date and time
- 🔊 Alarm notifications with sound alerts
- 😴 Snooze feature for reminders (5-minute delay)
- ✔ Mark tasks as completed / undo
- 🗑 Delete individual or all tasks
- 📊 Productivity progress tracker
- 🌙 Dark mode UI support
- 💾 Local storage (data persists after refresh)

---

## 🛠 Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Browser Notifications API
- LocalStorage API
